# Zurich Instruments MFLI 电压输入端故障维修状态说明

**项目目标**：管理并继续排查一台 Zurich Instruments MFLI 锁相放大器在 Signal Input +V 端过压后出现的电压输入异常问题。本文档用于 Codex / 工程协作，帮助后来接手者快速了解故障现象、已完成测量、当前判断和下一步实验安排。

**当前结论摘要**：故障已从“输入保护/前级泛化怀疑”收敛到 **AD9269BCPZ-65 B 通道上游的两级 ADA4932-1 差分驱动链路**。R61/R62 隔离后，AD9269 侧恢复为 0 V / 0 V 且仪器不再 OVI；异常 2.54 V / -0.72 V 保留在钳位/前级侧，并已追踪到下级 `ADA4932_2` Pin10/Pin11 输出经约 50 Ω 驱动钳位网络。继续上溯后，`ADA4932_1` 通过电阻/继电器网络强驱动 `ADA4932_2`；而 `ADA4932_1` 的前级 B8A / AD8512 输出保持约 1.26 V 平衡。最新 B8A 到 `ADA4932_1` 分段测量显示，`ADA4932_1 Pin3` 支路存在约 1 mA 量级拉低电流，当前首要嫌疑已收敛到 `ADA4932_1 Pin3` 输入/反馈局部网络或 `ADA4932_1` 本体。

---

## 1. 背景与故障现象

### 1.1 事故背景

设备：Zurich Instruments MFLI 锁相放大器。  
事故：Signal Input +V 电压输入端曾接入超过正常输入范围的电压，之后电压输入端出现异常。

### 1.2 初始症状

1. 电压输入端完全悬空时，点击自动量程，仪器跳转到 3 V 量程并报告输入超限 / OVI。
2. 在 3 V 量程下，LabOne Scope 无法观察到正常输入信号。
3. 手动切换到 1 V、100 mV 等小量程时，波形显示异常，并表现为超过当前量程的直流/噪声。
4. Signal Output 输出端正常，说明输出 DAC、部分数字控制与通信链路仍可工作。
5. 后续截图显示：1 V 量程约 1.05 V DC，100 mV 量程约 105 mV DC，说明显示值大致等于“当前量程 × 1.05”，更像是 ADC 原始码值接近满量程后，被软件按当前量程换算显示。

## 2. 重要手册信息

### 2.1 MFLI 输入规格要点

MFLI 电压输入正常测量量程最大为 ±3 V，但该量程不等于损坏阈值。根据 MFLI 手册，Voltage Input +V / -V Diff 的 damage threshold 为约 ±12 V；Current Signal Input I 的 damage threshold 为 ±5 V。因此，不能简单认为“电压输入超过 5 V 就必定损坏”。实际损坏取决于峰值、持续时间、源阻、地参考、50 Ω/10 MΩ 模式、尖峰和能量。

### 2.2 AD9269BCPZ-65 关键信息

```text
AD9269
BCPZ-65
```

## 4. 关键测量记录

### 4.1 输入保护器件

最初测得输入钳位器件低阻异常：

- 红框部分：红表笔接地，黑表笔分别点 6 个引脚，其中 3 个约 58 Ω、2 个约 28 Ω、1 个近 0 Ω。
- 绿框部分：对地不短路，两两之间基本为 28 Ω，偶尔 0 Ω。

处理：

- 拆除 4 个可疑钳位器件；
- 拆除后焊盘对地不短路；
- 但上电后 LabOne 显示仍异常。

结论：

- 保护器件可能确实受损；
- 但保护器件不是唯一故障点。

### 4.2 AD9269 A 通道输入

```text
Pin51 VIN+A = 0.88 V
Pin52 VIN−A = 0.88 V
Pin51 - Pin52 = 0 V
```

判断：

- A 通道共模约 0.88 V，接近 AD9269 典型输入共模 0.9 V；
- A 通道差分为 0 V；
- A 通道输入静态正常。

### 4.3 AD9269 B 通道输入

```text
Pin61 VIN−B = -0.72 V
Pin62 VIN+B = 2.54 V
Pin62 - Pin61 = 3.26 V
```

计算：

```text
B 通道共模 = (2.54 + (-0.72)) / 2 = 0.91 V
B 通道差分 = 2.54 - (-0.72) = 3.26 V
```

判断：

- B 通道共模约 0.91 V，看似正常；
- B 通道差分高达 3.26 V，远超 AD9269 满量程；
- Pin61 低于 -0.3 V，Pin62 高于 AVDD + 0.2 V，均超过 AD9269 输入绝对最大范围；
- B 通道处于严重过载状态，极可能触发 OVI。

### 4.4 AD9269 B 通道外围网络

用户绘制的 61/62 外围网络：

```text
Pin61 / Pin62
  │
  ├── 差分电阻 / 输入匹配
  │
  ├── 两路串联 R
  │
  ├── 差分 C
  │
  └── 钳位到 0 V 与 1.82 V 电源轨
```

实测特征：

```text
一路节点 = -0.72 V ≈ 0 V - 0.72 V
另一路节点 = 2.54 V ≈ 1.82 V + 0.72 V
```

判断：

- 这非常像输入钳位二极管正在导通；
- B 通道被某个前级或 ADC 本体异常推到了上下电源轨之外；
- 这是当前最关键的故障现象。

## 5. 当前故障判断

### 5.1 已基本排除或暂不优先怀疑

| 模块 | 状态 |
|---|---|
| Signal Output 输出链路 | 正常，不是当前主故障 |
| AD9269 A 通道输入 | Pin51/52 正常 |
| MCP23017 | 数字 I/O 扩展器，非 ADC，不优先 |
| B8A | 反馈结构对称，未直连 AD9269 VIN，暂不优先 |
| 单纯输入保护二极管 | 已拆除，焊盘不短路，但故障仍在 |

### 5.2 当前第一故障区域

```text
AD9269 B 通道输入链路：Pin61 / Pin62
```

### 5.3 当前两种主要假设

#### 假设 A：B 通道前级驱动器损坏或饱和

某个前级有源电路将 61/62 两路分别强行推向：

```text
-0.72 V
2.54 V
```

导致 B 通道输入保护钳位导通，最终使 AD9269 B 通道 overrange。

#### 假设 B：AD9269 B 通道输入级损坏

AD9269 B 通道内部输入保护、采样开关或输入结构被过压损坏，反向导致外部 61/62 网络被拉到异常电压。

### 5.4 当前更倾向的方向

由于 61/62 外围网络表现为上下轨外加二极管压降，现象更像 **B 通道输入对被外部或内部钳位结构强迫导通**。必须通过隔离实验判断源头，不应立即更换 AD9269，也不应继续盲目拆前级芯片。

## 6. 下一步实验计划

### 6.1 安全注意

当前 Pin61/62 已超过 AD9269 输入绝对最大范围。后续测试应遵循：

1. 不要外接任何输入信号。
2. 输入端保持短接或受控状态。
3. 每次上电只测必要点，测完立即断电。
4. 不要在通电状态下用电阻档/蜂鸣档。
5. AD9269 输入保护二极管已拆除/部分保护缺失，不要进行大信号注入。
6. 使用防静电操作，避免触碰 AD9269、LSK389、H2P 等敏感器件。

### 6.2 实验 1：测 ORA / ORB

目的：确认 OVI 是否由 B 通道触发。

| Pin | 名称 | 预期 |
|---:|---|---|
| 43 | ORA | A 通道 overrange |
| 22 | ORB | B 通道 overrange |

预期判断：

| 结果 | 判断 |
|---|---|
| ORA 无效、ORB 有效 | OVI 来自 B 通道，符合当前判断 |
| ORA 有效、ORB 无效 | 需重新确认通道映射 |
| 两者均有效 | ADC 参考/共模/时钟/全局状态可能异常 |
| 两者均无效但 LabOne OVI | OVI 可能由 FPGA/软件或其他检测路径产生 |

目的：确认 AD9269 全局参考与供电是否正常。

### 6.4 实验 3：比较 A/B 通道输入钳位器件

断电后，使用二极管档测：

- A 通道 51/52 对应的钳位器件；
- B 通道 61/62 对应的钳位器件。

记录：

```text
A 通道：节点到 0 V、节点到 1.82 V 的正反向压降
B 通道：节点到 0 V、节点到 1.82 V 的正反向压降
```

判断：

| 结果 | 意义 |

### 6.5 实验 4：隔离 61/62 输入串联电阻

这是最关键实验。

目标：区分“前级驱动异常”和“AD9269 B 通道输入自身异常”。

操作前提：

- 找准每一路进入 Pin61、Pin62 前的串联电阻；
- 不要拆 61/62 之间的差分电阻；
- 不要拆差分电容；
- 只隔离“前级节点 → 串联 R → AD9269 Pin61/Pin62”中的两颗串联 R。

实验：

1. 断电；
2. 将 Pin61 对应串联电阻翘起一端；
3. 将 Pin62 对应串联电阻翘起一端；
4. 上电短时间测：
   - 前级侧 61 路节点；
   - 前级侧 62 路节点；
   - AD9269 Pin61 侧；
   - AD9269 Pin62 侧；
   - ORB 状态。

判断：

| 结果 | 结论 |
|---|---|
| 前级侧仍为 -0.72 / 2.54 | 前级驱动器损坏或饱和 |
| 前级侧恢复正常，ADC 侧异常 | AD9269 B 通道输入级损坏 |
| ADC 侧仍自发出现 -0.72 / 2.54 | AD9269 B 通道内部损坏高度可疑 |
| 隔离后 ORB/OVI 消失 | 前级是异常源 |
| 隔离后 ORB/OVI 仍在 | ADC B 通道或数字判定异常 |

### 6.6 实验 5：反向追踪 B 通道前级源头

如果隔离实验表明前级侧仍异常，需要断电追线：

```text
2.54 V 节点 → 上游有源芯片输出
-0.72 V 节点 → 上游有源芯片输出
```

重点查：

- 是否连接到某颗双运放/驱动器；
- 是否经过继电器/模拟开关；
- 是否连接到 H2P、B8A、LTFSZ 或其他未识别芯片；
- 是否有钳位器件直接损坏。

## 10. 工程备注

1. AD9269 为 64-lead LFCSP，带裸露焊盘，手工更换难度较高。
2. 如果更换 AD9269，应使用热风/预热台，并注意底部焊盘。
3. 更换 ADC 后可能需要仪器自校准或至少进行全量程验证。
4. 在最终修复前，不应向 Signal Input +V 施加任何外部信号。
5. 输入保护器件应在最终恢复时更换/补回，否则下次测试风险极高。
6. 若需要临时验证 ADC，可在隔离输入后用高阻方式将 VIN+B/VIN−B 同时偏置到 VCM，再观察 ORB/OVI 是否消失；该操作需要非常谨慎，避免直接短接或过驱 ADC 输入。
